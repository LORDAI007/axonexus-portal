
# Axonexus Portal

Portal institucional del Imperio de la Gobernanza Predictiva (Cloudflare Pages).

## Estructura
- `index.html`: landing principal
- `sections/*`: páginas internas (Core, Prensa, Marco Legal, Universidad, Contacto)
- `styles/main.css`: tema negro/dorado
- `assets/logo-axonexus.svg`: emblema

## Deploy
Este repo se despliega automáticamente en Cloudflare Pages.
